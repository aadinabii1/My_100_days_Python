# 🐍 My 100 Days of Code - Python Journey

This repository documents my progress through **[100 Days of Code™: The Complete Python Pro Bootcamp](https://www.udemy.com/course/100-days-of-code/)** by Dr. Angela Yu.

Each day, I build a small project to practice a new Python concept — starting from the basics and working up to more advanced topics like web scraping, APIs, automation, and web development.

## 📂 Repository Structure

Projects are organized by day:

```
My_100_days_Python/
├── Day 1/
├── Day 2/
├── Day 3/
├── Day 4/
│   └── Rock Paper Scissors/
├── Day 5/
└── ...
```

Each project folder contains its own code and, where relevant, a README explaining what the project does and what I learned.

## 🎯 Goals

- Build consistency by coding every day
- Strengthen my Python fundamentals
- Get hands-on practice with real, working projects
- Track my growth as a beginner developer over time
